Silver Thoughts

A simple app where you can write short posts and read others'. It uses React and Vite.

What the app uses

- React and Vite (fast development)
- Tailwind CSS for styling
- Firebase for login and storing posts (see `src/Config`)

How to set up Firebase

1. Put your Firebase settings into `src/Config/firebase-config.tsx`.
2. Restart the dev server.

Important files location

- Pages: `src/Pages`
- Firebase settings: `src/Config`
- Auth helpers: `src/LoginStateTracker`
